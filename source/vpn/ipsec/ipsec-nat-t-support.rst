IPsec NAT-T Support
===================

Yes, NAT Traversal for IPsec (NAT-T) is supported in all current
versions. It is configured on the Phase 1 options for an IPsec tunnel.

.. seealso:: For more information on NAT traversal, see the
   `Choosing IPsec configuration options`_ page in The pfSense Book.
   
.. _Choosing IPsec configuration options: /pfsense/en/latest/book/ipsec/choosing-configuration-options.html#nat-traversal
